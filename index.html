Orion: A Semantic Uplift Pipeline for Building Persistent, Queryable Knowledge from Egocentric Video

Riddhiman Rana*1 Aryav Semwal1 Yogesh Atluru1 Kevin Zhu1† Sunith Vallabhaneni1†

Paper PDF Code Cite
Abstract

Understanding causal dynamics in egocentric video remains a fundamental challenge for grounded language and vision systems. Existing approaches detect objects and actions but fail to synthesize these into coherent, causally structured representations.

We introduce Orion, a system that transforms raw perceptual streams into high-level, symbolic knowledge graphs — a process we term semantic uplift. Orion integrates state-of-the-art perception and reasoning components, including object detection (YOLOv11), segmentation (SAM), object description (FastVLM), embedding-based tracking (CLIP), semantic label correction (Sentence-BERT), and high-level event summarization (Gemma 3), collectively constructing temporally aligned, dynamic knowledge graphs from raw video.

Using the Action Genome dataset, Orion achieves strong relation prediction accuracy and causal coherence. Our key contribution is a scientifically grounded system supported by causal scoring, configuration-aware thresholds, and constrained prompt-based reasoning — positioning semantic uplift as a bridge between low-level vision and high-level causal language representations.

  • ⚡

Method

Semantic Uplift Pipeline

Orion processes raw egocentric video through five sequential stages, bridging low-level visual signals to persistent, causally structured knowledge graphs.

01 / PERCEPTION
Perception Phase
YOLOv11x detection · CLIP embeddings · Spatial analysis
02 / TRACKING
Semantic Uplift
HDBSCAN clustering · State detection · LLM event composition
03 / GRAPH
KG Construction
Entity nodes · Causal reasoning · Temporal sequencing
04 / STORAGE
Storage & Index
Neo4j persistence · Vector indexing · Relation constraints
05 / QUERY
Query & Q&A
Knowledge retrieval · LLM-based answer generation

👁️ Dual-Loop Perception

A high-frequency loop (30–60 FPS) handles frame sampling and visual embedding extraction via CLIP, while a slower loop (1–5 FPS) manages rich FastVLM descriptions — decoupling perception throughput from LLM cost.

🔗 Entity Tracking

HDBSCAN clustering on L2-normalized CLIP embeddings (512–2048 dim) assigns stable entity IDs across occlusion and re-identification. Optical flow and velocity support temporal state linking.

✅ Semantic Validation

A hybrid rule-based + Sentence-BERT layer (all-MiniLM-L6-v2) corrects COCO misclassifications. Classes with similarity < 0.75 are re-evaluated. Improves accuracy from ~80% to 85–90%.

⚗️ Causal Inference

A Causal Influence Score (CIS) combines temporal ordering, spatial proximity, and motion/semantic correlation to propose directed causal links between state-change event pairs.
